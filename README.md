## üèÅ Getting Started <a name = "getting_started"></a>

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See [deployment](#deployment) for notes on how to deploy the project on a live system.

### Prerequisites

To build the docker image

```
docker build -t stock-analysis .
docker images
docker run -d -p 8080:8000 stock-analysis
```

Check processes in docker and get docker id
```
docker ps
```

to run terminal of docker on the system
```
docker exec -it stock-analysis /bin/bash
```

send request to server
```
curl -XPOST -F 'username=**your_username**' -F 'password=**your_password**' http://localhost:8080/api-token-auth/
```

Create new record
```
curl -X POST -H 'Content-Type: application/json' -H 'Authorization: Token **your_token**' -d '{"item": "**your_item_uuid**", "quantity": "1"}' http://api:8000/order/
```

Retrieve all orders
```
curl -X GET -H 'Authorization: Token **your_token**' http://api:8000/order/
```
